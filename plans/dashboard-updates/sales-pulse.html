Design Notes
Hero: The momentum percentage IS the headline — not a conversion count. "↑ 23%" with "Accelerating" badge tells the story instantly. Color-coded green for accelerating, red for decelerating.

Context line: Shows the raw order counts behind the percentage so the user trusts the number and can feel the scale.

Sparkline: Daily order count over the full 28-day window (prior 14 + last 14). A dashed vertical line marks the midpoint so the user can visually compare the two halves. The rising right side IS the acceleration.

"What's Driving It": Breaks the momentum into its two components — order count trend and average order value trend. This answers "are we getting more customers or are customers spending more?" Both can be true independently.

14-Day Revenue: The bottom summary grounds everything in dollars. The 28% combines both drivers (more orders × higher AOV).

Rolling window: Uses 14 vs. prior 14 days for a smoother signal than 7-day comparisons. Recalculates daily.
