Design Notes
Hero: True CAC dominates — the first thing the eye hits.

Comparison: Platform-reported numbers are visually muted, positioned as "claims" the user can evaluate against Shopify reality.

Reality row: Indigo-highlighted to reinforce this is the number that matters.

Over-claim bar: Makes the gap tangible without requiring math. Amber signals caution, not crisis.

Adapts to platforms: Only shows rows for platforms the client has connected. One platform? One row. Three? Three rows.
